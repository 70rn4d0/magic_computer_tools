## 🚀 How to Use

### Windows Users
1. Download the `.exe` file
2. Double-click to run
3. Follow the simple GUI

### Linux Users
1. Download the source code
2. Compile: `g++ -o toolname toolname.cpp`
3. Run: `./toolname`
